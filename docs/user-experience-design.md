# 2AMI 用户体验设计

## 产品核心

**让用户把"会做的事"变成收益。**

```
你会做一件事 → 教给 2AMI → 分享出去 → 别人用了你就赚钱
```

---

## 用户界面

主界面只有三个入口：

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│     [让 AMI 帮我做]    [教 AMI 新技能]    [我的收益] │
│                                                     │
└─────────────────────────────────────────────────────┘
```

---

## 体验流程

### 流程 1：让 AMI 帮我做

**用户只需要说想做什么，不需要关心怎么做。**

```
┌─────────────────────────────────────────────────────┐
│ 你想做什么？                                        │
│                                                     │
│ ┌─────────────────────────────────────────────────┐ │
│ │ 帮我在小红书发一篇笔记                          │ │
│ └─────────────────────────────────────────────────┘ │
│                                        [开始]       │
└─────────────────────────────────────────────────────┘
```

执行中（简洁，不暴露技术细节）：

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  ⏳ 正在帮你发布...                                 │
│                                                     │
│  ✓ 打开小红书                                       │
│  ✓ 进入发布页面                                     │
│  ○ 填写内容并发布                                   │

完成：

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  ✅ 已发布                                          │
│                                                     │
│  笔记已成功发布到小红书。                           │
│                                                     │
│  [查看笔记]                                         │
│                                                     │

## 设计原则

1. **零学习成本** - 用户不需要理解任何技术概念
2. **即时反馈** - 每一步操作都有清晰的状态
3. **收益可见** - 让用户清楚看到自己的贡献带来了多少收益
4. **信任感** - 需要敏感操作时（如登录），明确告知用户

## 关键交互细节

### 当 AMI 不会做时

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  🤔 这个我还不会                                    │
│                                                     │
│  我还没学过怎么做这件事。                           │
│                                                     │
│  [教我怎么做]    [算了]                             │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 当执行失败时

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  😅 没成功                                          │
│                                                     │
│  网页结构可能变了，我的方法不管用了。               │
│                                                     │
│  [再试一次]    [我来教你新方法]                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 发现已有类似技能时

```

│  💡 发现类似技能                                    │

│  已有人分享过类似的技能：                           │

│                                                     │
│  [直接用这个]    [我的方法不一样，继续教]           │
│                                                     │
└─────────────────────────────────────────────────────┘
```
